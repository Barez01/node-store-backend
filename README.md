# Notefy App (Server)

A High level React & Node JS application built for managing daily tasks. Written in Typescript & Javascript with full clean code and architecute.

## Features
- Built with Node JS
- Clean Architecute
- MySql database used
- Basic CRUD operations
- Basic API operations with express and axios
- Token creation and verifying

## Installation

1. **Clone the repository**  
   ```sh
   git clone https://github.com/Barez01/node_tasks.git
2. **Navigate to the project directory**  
   ```sh
   cd node_tasks
3. **Install dependencies**  
   ```sh
   npm install
4. **Start the development server**  
   ```sh
   node server.js or npm run dev
4. **Use the web app**  
   ```sh
   get access through: https://github.com/Barez01/react-task-manager.git
